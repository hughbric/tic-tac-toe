## Tic Tac Toe

I built this to explore:
- Next.js
- Vercel
- TailwindCSS
- React Hooks
- React Testing Library

### How to use
Clone the repository and navigate to the project root.

Make sure you're using one of the latest versions of Node:
`nvm use 16.0.0`

Run:
`npm install`

To run the dev server:
`npm run dev`

To run the tests:
`npm test`
